Системный аналитик
Разработка сервиса бронирования для компьютерных клубов
Описание:
• Анализ бизнес-требований: Сбор и анализ требований от заказчиков, определение целей и задач проекта, формирование продуктового видения.
• Проектирование системы: Разработка архитектуры системы, проектирование модулей и компонентов, создание UML-диаграмм.
• Работа с базами данных: Написание запросов с выводом в дашборде для мониторинга бизнес показателей
• Проектирование интерфейсов: Создание прототипов с помощью draw io/figma.
• Документирование: Создание технической документации для системы, включая описание архитектуры, модулей, API и инструкций для пользователей.
• Поддержка и сопровождение: Оказание технической поддержки сервиса, решение проблем пользователей, доработка системы по мере необходимости.
Достижения:
• Разработка и внедрение сервиса для компьютерных клубов, который используется более чем в 100 клубах по всей стране.
• Реализация функций бронирования компьютеров, тарифов, таймеров, дашборда и личного кабинета клиента.
• Повышение удовлетворенности клиентов за счет удобного и функционального сервиса.
Используемые технологии:
• Бэкенд: PHP (Laravel), PostgreSQL, Redis
• Фронтенд: Vue.js
• Десктопное приложение: Electron.js
